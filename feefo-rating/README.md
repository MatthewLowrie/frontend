## How to Run the Program

make sure you are in feefo-rating:

cd feefo-rating

### 1. **Install Dependencies**

Make sure you have [Node.js](https://nodejs.org/) installed (recommended version: 18+).

Install all required dependencies by running:
npm install

### 2. **Run Dev Server**

Start Development server by running:
npm run dev

### 3. **Run Unit Tests**

Run all unit tests by running:
npm test
